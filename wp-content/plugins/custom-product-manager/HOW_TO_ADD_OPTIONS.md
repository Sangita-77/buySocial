# How to Add Options to Sub-Categories

## Step-by-Step Guide

### Step 1: Add a Category
1. Go to **Custom Products** â†’ **Add Product** (or edit existing)
2. Click **"Manage Categories, Sub-Categories & Variations"**
3. Click **"Add Category"** button
4. Enter category name (e.g., "Buy Facebook Likes")

### Step 2: Add a Sub-Category
1. Under your category, click **"Add Sub-Category"** button
2. Enter sub-category name (e.g., "Select Facebook Like Type")
3. Select type: **"Dropdown"** (this is selected by default)
4. You will see an **"Options"** section appear below

### Step 3: Add Options
1. In the Options section, click **"Add Option"** button
2. Enter your first option (e.g., "Facebook Page")
3. Click **"Add Option"** again to add more options:
   - "Facebook Post"
   - "Facebook Video"
   - "Facebook Story"
   - etc.

### Step 4: Continue Adding Sub-Categories
Add more sub-categories as needed:
- "Select Target Country" (Dropdown type)
  - Options: USA, UK, Canada, etc.
- "Select Quantity" (Number type - no options needed)

### Step 5: Add Variations with Prices
After adding sub-categories, you can:
1. Click **"Add Variation/Price"**
2. Select values for each sub-category
3. Enter the price
4. Click **"Save All Categories"**

## Example Structure

**Category:** Buy Facebook Likes

**Sub-Category 1:** Select Facebook Like Type (Dropdown)
- Option: Facebook Page
- Option: Facebook Post
- Option: Facebook Video

**Sub-Category 2:** Select Target Country (Dropdown)
- Option: USA
- Option: UK
- Option: Canada

**Sub-Category 3:** Select Quantity (Number)

**Variations:**
- Facebook Page + USA + 1000 = $10.00
- Facebook Post + UK + 5000 = $45.00
- etc.

## Tips

- **Dropdown type** = Shows options section where you can add multiple choices
- **Number type** = Customer enters a number (no options needed)
- **Text type** = Customer enters text (no options needed)
- You can add as many options as needed
- Click "Remove" next to any option to delete it
- Make sure to click "Save All Categories" when done
